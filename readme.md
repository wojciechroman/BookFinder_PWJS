# BookFinder - Projekt w Języku Skryptowym

## 1. Opis
Aplikacja typu BookFinder, pozwalająca wyszukiwać literaturę. Aplikacja oparta wykorzystuje API Google Books.

## 2. Wykorzystana Technologia
* JavaScripi
* HTML
* CSS
* API Google Books
